QTopo - Drawing


In the QTopo drawing window you can draw scraps and save them in therion format "th2".

To effectively draw your scraps stick to the following workflow:

  1. check in the shot table that the shot stations are correct. You can use the 3D window and the plan and extended section windows to visualize the centerline with the stations. The plot do not update at the changes to the shot table; to refresh them switch on and off the grid.
  2. check with the extended plot window that the shots are extended correctly. These two steps can be done together.
  3. draw the plan and the extended section scrap. It is useful (but not necessary) to follow these steps:
    • select the station points;
    • draw the wall lines;
    • draw the other lines;
    • draw the points.
  4. save the scraps
